# E-commerce Price Tracker 🛒

**Purpose:** Track product prices, names, and ratings across multiple pages.  
**Skills:** Multi-page scraping, data cleaning, Excel export  
**Output:** Excel file with Product Name, Price, Rating  

**How to Run:**
1. Install Python libraries: `requests`, `beautifulsoup4`, `pandas`
2. Run `price_tracker.py`
3. Check `Book_Price_Data.xlsx` for output

**Sample Output Screenshot:** `output_screenshot.png`

**Freelancing Use:** Online sellers, dropshippers, marketers — competitor price monitoring.
