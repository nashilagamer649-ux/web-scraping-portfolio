# Job Listings Scraper 💼

**Purpose:** Collect job postings with title, company, location, and apply link.  
**Skills:** Web scraping, multi-field extraction, Excel export  
**Output:** Excel file with Job Title, Company, Location, Apply Link  

**How to Run:**
1. Install Python libraries: `requests`, `beautifulsoup4`, `pandas`
2. Run `job_scraper.py`
3. Check `Job_Listings_Data.xlsx` for output

**Sample Output Screenshot:** `output_screenshot.png`

**Freelancing Use:** HR agencies, job portals, recruiters — fresh job data collection.
