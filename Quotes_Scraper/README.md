# Quotes Scraper 📝

**Purpose:** Extract inspirational quotes and authors from websites.  
**Skills:** HTML parsing, Python loops, CSV export  
**Output:** CSV file with Quote Text + Author Name  

**How to Run:**
1. Install Python libraries: `requests`, `beautifulsoup4`, `pandas`
2. Run `quotes_scraper.py`
3. Check `quotes_data.csv` for output

**Sample Output Screenshot:** `output_screenshot.png`

**Freelancing Use:** Bloggers, motivational apps, social media content creators.
