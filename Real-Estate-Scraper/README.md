# Real Estate Scraper 🏠

**Purpose:** Scrape property listings with title, price, area/description, and link.  
**Skills:** Structured data extraction, multiple field scraping, Excel reporting  
**Output:** Excel file with Property Title, Price, Area/Details, Link  

**How to Run:**
1. Install Python libraries: `requests`, `beautifulsoup4`, `pandas`
2. Run `real_estate_scraper.py`
3. Check `Real_Estate_Listings.xlsx` for output

**Sample Output Screenshot:** `output_screenshot.png`

**Freelancing Use:** Real estate agents, investors — market research and comparison.
