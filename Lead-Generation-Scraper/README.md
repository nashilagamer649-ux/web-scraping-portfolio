# Lead Generation Scraper 📊

**Purpose:** Extract company details from business directories — name, contact info, website.  
**Skills:** Directory scraping, contact data collection, Excel automation  
**Output:** Excel file with Company Name, Contact, Address, Website Link  

**How to Run:**
1. Install Python libraries: `requests`, `beautifulsoup4`, `pandas`
2. Run `lead_generator.py`
3. Check `Business_Leads_Data.xlsx` for output

**Sample Output Screenshot:** `output_screenshot.png`

**Freelancing Use:** Marketing agencies, sales teams — ready-to-use leads.
